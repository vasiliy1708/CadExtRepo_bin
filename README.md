# CadExtRepo_bin
Модуль расширения для Evolution.

Систеные требования: версия SQL Server 2017 и выше.

Установка: 
  1. Скопировать dll в папку C:\Evolution\eCadPro
  2. Зарегистрировать командой в терминале "%WINDIR%\Microsoft.NET\Framework\v4.0.30319\regasm.exe" С:\Evolution\eCadPro\CadExtRepo.dll /tlb /nologo /codebase
